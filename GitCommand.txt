sudo systemctl disable nginx 2>/dev/null

# 3. Kiểm tra port 80 đã free chưa

Nếu bạn muốn “get đè” (lấy lại code mới nhất và ghi đè lên thư mục đang có), có 2 cách phổ biến:
Cách 1 (khuyến nghị): Reset sạch về đúng trạng thái remote (mất mọi sửa đổi local)
Chạy trong thư mục repo:
cd /duong/dan/toi/okr-web

git fetch --all
git reset --hard origin/main   # nếu nhánh chính là main
git clean -fd                  # xóa file/thư mục chưa được git theo dõi

Nếu repo dùng nhánh master thì đổi origin/main thành:
git reset --hard origin/master

Bạn có thể xem nhánh đang theo dõi là gì bằng:
git branch -vv

Cách 2: Xóa thư mục cũ rồi clone lại (đơn giản, chắc chắn)
Ở thư mục cha (parent):
rm -rf okr-web

Nếu bạn muốn giữ lại một phần thay đổi local (không muốn mất hết), nói mình biết bạn có cần giữ gì không (ví dụ .env, file config), mình sẽ hướng dẫn cách backup/stash trước khi “get đè”.
